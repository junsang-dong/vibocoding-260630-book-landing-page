AI 시대 개발법

코드를 외우는 시대는 끝났습니다. AI와 협업하며 문제를 해결하는 새로운 개발 방식을 익힙니다.

⌨️

Cursor 활용

AI 코드 에디터 Cursor를 200% 활용해, 입문자도 빠르게 결과물을 만들어 냅니다.

🧠

Claude 활용

강력한 LLM Claude로 설계, 디버깅, 리팩터링까지. 든든한 페어 프로그래머와 함께합니다.

🛠️

실전 프로젝트

이론이 아닌 실전. 30개의 진짜 동작하는 앱을 손으로 직접 만들며 배웁니다.

Learning Journey

학습 로드맵

30일, 한 걸음씩 성장하는 항해 코스

  1. DAY 1

    Landing Page

    HTML/CSS/JS 기초로 첫 웹페이지를 완성합니다.

  2. DAY 5

    API

    외부 API를 연동하고 비동기 데이터를 다룹니다.

  3. DAY 10

    React

    컴포넌트 기반 모던 프론트엔드 개발에 입문합니다.

  4. DAY 20

    AI Agent

    RAG, MCP를 활용한 AI 에이전트를 직접 구축합니다.

  5. DAY 30

    Deployment

    완성한 앱을 세상에 배포하며 항해를 마칩니다.
